# netflix-frontend-clone

## Live Demo:
$ https://netflix-frontend-clone.herokuapp.com/
## :point_down:Steps to initialize the project:

- Clone the repository
```
$ git clone https://github.com/anjupriya-v/netflix-frontend-clone.git
```
- Redirect to the root directory of cloned repo 

- Start the Live Server in Visual Studio Code or Open the index.html File for Demo.

## :point_down:Steps to host this project on Heroku:

- Open up the terminal and redirect to the project directory

- Execute the Following Commands
```
heroku login
```

```
heroku create projectName
```

```
git init
```
```
heroku git:remote -a projectName
```
```
git add .
```
```
git commit -am "make it better"
```
```
git push heroku master
```
